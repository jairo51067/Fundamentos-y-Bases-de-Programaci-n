# ⚡ JavaScript Fundamentals: Notas y Práctica

Una documentación interactiva y repositorio de prácticas basado en los fundamentos de JavaScript, siguiendo la guía de [LenguajeJS](https://lenguajejs.com/fundamentos/). Este proyecto sirve como bitácora de aprendizaje y referencia técnica.

---

## 🚀 Contenido del Proyecto

Este repositorio cubre los pilares esenciales del lenguaje, organizados de manera lógica para facilitar su consulta:

* **Introducción a JS:** Historia, sintaxis básica y "Hola Mundo".
* **Variables y Tipos de Datos:** Let, const, strings, números y booleanos.
* **Estructuras de Control:** Condicionales (if, switch) y bucles (for, while).
* **Funciones:** Declaración, expresiones y arrow functions.
* **Objetos y Arrays:** Manipulación de datos y métodos esenciales.
* **DOM:** Interacción básica con el navegador.
